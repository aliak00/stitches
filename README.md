# Stitches

## Development

### Install

```sh
# yarn
yarn

# npm
npm install
```

### Build

```sh
# yarn
yarn build

# yarn (watch)
yarn build:watch

# npm
npm run build

# npm (watch)
npm run build:watch
```

### Test

```sh
# yarn
yarn test

# yarn (watch)
yarn test:watch

# npm
npm test

# npm (watch)
npm run test:watch
```
